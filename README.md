# Visualization for N-Queens Problem

This repository contains a visualization tool for the N-Queens problem. The N-Queens problem is a classic problem in the field of computer science and combinatorial optimization, where the goal is to place N chess queens on an NÃ—N chessboard in such a way that no two queens threaten each other. In this repository, we provide a visualization tool to help understand various solutions and algorithms for solving the N-Queens problem.

## Contents

1. **`index.html`**: HTML Frame for visualization tool.
2. **`n-queens.js`**: Solution in JS
3. **`README.md`**: This file provides an overview of the repository and instructions on how to use the visualization tool.
